The playtester loop

Build → solo goldfish → tweak → solo goldfish → invite a friend → matchup test → tweak → matchup test. Then repeat with a different opponent's deck to test a wider matchup spread. TCG Playsim is built around making each step as fast as possible: the deckbuilder has live Scryfall search, decks save automatically, hotkeys speed solo testing, and rooms launch in seconds.

Playtest any format

Standard, Commander (EDH), Oathbreaker, Modern, Pioneer, Pauper, Legacy, Dandân, and any homebrew or Cube draft format. Format is set per-deck. The playtester doesn't enforce legality (you do), so you can test cards from any printing, any era, against any other deck.

Playtest solo (free, no account)

Single-seat games run with no account. Build a deck, click Play, you're in. Mulligan, draw, play turns, hit the F key to search any zone, U to untap, D to draw — the full Magic gameplay loop. The action log keeps a running record so you can review the play sequence.

Playtest with friends (free, account needed for hosting)

Sign in (Google or magic-link), click Multiplayer → Host Room, share the link. Up to 4 players. Real-time multiplayer over websocket, low-latency. Public rooms are also visible in the multiplayer lobby for finding strangers to test against.
