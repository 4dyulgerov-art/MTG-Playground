MTG Arena vs. TCG Playsim — la comparaison honnête

MTG Arena est un jeu digital automatique poli, avec un art magnifique, un moteur de règles complet, du jeu classé, du draft, et une économie de cartes. C'est brillant pour ce que c'est. Les compromis : vous farmez les cartes (ou vous payez), la liste de formats est courte (Standard, Historic, Brawl, Alchemy, Explorer, Timeless), et vous ne pouvez pas jouer Commander, Oathbreaker, Pauper, ou aucun format homebrew sans bidouilles tierces.

TCG Playsim est le compromis inverse. Le moteur de règles est manuel — vous et votre adversaire résolvez les interactions vous-mêmes, comme à une table — mais en échange vous obtenez toutes les cartes dans Scryfall, tous les formats y compris Commander et Dandân, des parties à 2-4 joueurs, et zéro économie. Tourne dans n'importe quel navigateur moderne sans téléchargement. C'est gratuit.

Quand Arena est l'outil approprié

Choisissez MTG Arena si vous voulez du solo poli, du Standard / Limited classé, des drafts contre bots ou queues, vérification automatique des règles, et que vous ne voulez pas farmer (ou payer) les cartes. L'économie est le prix du polish.

Quand TCG Playsim est l'outil approprié

Choisissez TCG Playsim si vous voulez jouer Commander, Oathbreaker, Pauper, Dandân, ou n'importe quel format homebrew ; si vous voulez toutes les cartes immédiatement disponibles pour test ; si vous voulez jouer avec des amis dans une partie privée gratuite ; si vous voulez une table virtuelle plutôt qu'un jeu automatique ; ou si vous ne voulez rien installer.
