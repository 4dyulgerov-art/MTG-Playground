Qu'est-ce qu'Oathbreaker ?

Oathbreaker est un format 60 cartes singleton avec un planeswalker comme « oathbreaker » (équivalent commandant) plus un sort signature — un instant ou rituel qui ne peut être lancé qu'avec le planeswalker en jeu. Les deux démarrent dans la zone de commandement. Decks singleton ; identité de couleur du planeswalker. PV de départ : 20 (vs. les 40 de Commander), ce qui rend les parties plus courtes et la loyauté du planeswalker importante.

Le format est petit mais dévoué. Il y a peu d'endroits pour y jouer en ligne — la plupart des clients le traitent comme une variante Commander avec setup manuel, ce qui marche mais c'est inélégant.

Oathbreaker dans TCG Playsim

Support de premier ordre : le deckbuilder a une option de format Oathbreaker qui sépare un slot planeswalker ET un slot sort signature. Les deux démarrent dans la zone de commandement. Le playtester sait que le sort signature ne peut être lancé que pendant que l'oathbreaker est en jeu (vous l'enforcez vous-même — le playtester est manuel — mais le layout visuel rend ça évident).

Construire un deck Oathbreaker

Nouveau Deck → format : Oathbreaker → choisir le planeswalker → choisir le sort signature → construire les 58 autres cartes. Identité de couleur du planeswalker. Règles singleton (un de chaque non-basique). La plupart des decks penchent vers tutorer le sort signature, protéger l'oathbreaker, ou les deux.

Oathbreaker multijoueur

Le format se joue proprement à 2-4 joueurs. Chaque joueur a son propre oathbreaker + sort signature. Free-for-all par défaut ; la politique fait partie du méta.
